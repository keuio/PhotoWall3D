# PhotoWall3D
一个基于 **GSAP** 动画库开发的 3D 照片墙组件。它能将一组图片自动排布成一个 3D 球体，支持自动旋转、鼠标/触摸拖拽交互以及点击放大查看。（A high-performance 3D photo wall component built with **GSAP**. It automatically arranges a collection of images into a 3D sphere, supporting auto-rotation, mouse/touch drag interaction, and a smooth "click-to-zoom" feature.）
